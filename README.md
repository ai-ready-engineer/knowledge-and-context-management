# Knowledge and Context Management

Course materials for the AI-Ready Engineer masterclass on Knowledge and Context Management.

Part of the [AI-Ready Engineer](https://ai-ready-engineer.github.io/) series. Sibling course: Experiment Design and Evaluation of AI Systems.

## Structure

- `index.html` — course homepage (the public landing page)
- `style.css` — shared AI-Ready Engineer theme
- `SYLLABUS.md` — course narrative and lesson list
- `lesson_N/` — one folder per lesson, each with:
  - `LN_outline.md` — what we cover, skills unlocked, tools (the public-facing summary)
  - `LN_details.md` — concepts, patterns, anti-patterns, worked examples, the hands-on example
  - `LN_script.md` — talk outline / beats / demo walkthrough for delivering the lesson

## Lessons

Lifecycle spine: *represent → connect → extract & discover → query → provide context → improve*, with quality of knowledge as the through-line.

1. Quality of Knowledge
2. Knowledge Representation
3. Knowledge Graphs
4. Knowledge Extraction & Discovery
5. Knowledge Querying
6. Providing Context to Agents
7. Knowledge Improvement & Agentic Feedback
