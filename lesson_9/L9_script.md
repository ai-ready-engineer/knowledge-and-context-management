# Script — L9 Knowledge Improvement

> Talk outline / script. Beats to hit, transitions, what to show on screen, what to say at each turn.

## Open

Show the L1 scorecard from week one — all green. Then show it three months later, untouched, quietly red: a policy changed, gaps opened, two sources now contradict, and agents have been failing on refunds for weeks. "Nobody broke it. It decayed — and your agents have been telling you the whole time. The signals were there. Were they mined?"

Hook: "Every other lesson built, served, or used the knowledge. This one keeps it from dying — and turns the whole lifecycle into a loop."

## Beats

1. Knowledge decays: staleness, gaps, contradictions, drift. Name each; point back to where we saw it (L3 conflicts, L5 "not found", L6 predicted gaps, L8 agent dead-ends).
2. The loop: detect → prioritize → fix → verify, scored against the L1 scorecard. The spine of the lesson.
3. Signals are the fuel — and we now have a whole course of them: L5 unanswered, L6 discovery, L7 context failures, L8 agentic backprop. A ranked, evidence-based backlog. Opinions are not.
4. AI as KM co-worker — detect contradictions, propose updates, fill gaps. Then the hard turn: this is where it gets dangerous.
5. Governance — the gate. Provenance (L4) + human-in-the-loop, proportional to impact, *and* the L8 write-back gate. AI and agents propose; the loop disposes.

## Demo / playground walkthrough

- Aged knowledge base + the signal streams: unanswered (L5), discovery (L6), context failures (L7), agentic backprop (L8). Run detect: contradictions and gaps surface; L8 failures trace via provenance to the bad source.
- Prioritize by traffic × impact; fix the top items (retire stale, add gap doc, resolve conflict, accept a grounded agent proposal through the gate); re-run the gold questions; scorecard recovers — *measured*, not asserted.
- The cautionary demo: auto-accept ungated agent write-back → a refund window silently flips 30→45. Then show the gate + provenance catching it. Land the governance point hard.
- Final slide: the lifecycle loop closes — represent → connect → extract → query → discover → serve context → backpropagate → improve → back to represent.

## Close

Single takeaway: **Knowledge management is a loop, not a project. Drive it with the signals from querying, discovery, context, and your agents; fix at the right stage; verify against the scorecard; and gate every write — especially the agent's. The same AI and agents that use your knowledge can keep it true — only with provenance and gates in the loop.**

## Notes to self

- This is the capstone — explicitly tie every beat back to L1–L8 so the course lands as one connected lifecycle.
- The ungated-write-back demo is the moral climax; make the silent number-change subtle so the room feels how easily it slips by. It's the same trap as L8's contamination, now at the KB level.
- End on the loop diagram: represent → connect → extract → query → discover → serve → backpropagate → improve → back to represent.
