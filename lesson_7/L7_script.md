# Script — L7 Knowledge Improvement

> Talk outline / script. Beats to hit, transitions, what to show on screen, what to say at each turn.

## Open

Show the L1 scorecard from week one — all green. Then show it three months later, untouched, quietly red: a policy changed, gaps opened, two sources now contradict. "Nobody broke it. It decayed. Knowledge is alive, and alive things rot if you don't tend them."

Hook: "Every other lesson built or used the knowledge base. This one keeps it from dying — and turns the lifecycle into a loop."

## Beats

1. Knowledge decays: staleness, gaps, contradictions, drift. Name each; point back to where we saw it (L3 conflicts, L5 "not found", L6 predicted gaps).
2. The loop: detect → prioritize → fix → verify, scored against the L1 scorecard. The spine of the lesson.
3. Signals are the fuel — unanswered queries (L5) and discovery findings (L6) are a ranked, evidence-based backlog. Opinions are not.
4. AI as KM co-worker — detect contradictions, propose updates, fill gaps, summarize. Then the hard turn: this is exactly where it gets dangerous.
5. Governance — provenance (callback to L4) + human-in-the-loop, proportional to impact. AI proposes; human approves the writes that matter.

## Demo / playground walkthrough

- Aged knowledge base + a log of usage / unanswered (L5) / thumbs-down / discovery findings (L6). Run detect: contradictions and gaps surface, failures trace via provenance to the bad source.
- Prioritize by traffic × impact; fix the top three (retire stale, add gap doc, resolve conflict); re-run the gold questions; scorecard recovers — *measured*, not asserted.
- The cautionary demo: let the LLM "improve" articles in bulk, unreviewed → a refund window silently flips 30→45. Then show provenance + a review gate catching it. Land the governance point hard.
- Final slide: the lifecycle loop closes — represent → connect → extract → query → discover → improve → back to represent.

## Close

Single takeaway: **Knowledge management is a loop, not a project. Drive it with signals, fix at the right stage, verify against the scorecard, and keep a human on the writes that matter. The same AI that reads your knowledge can help you keep it true — only if you keep provenance and review in the loop.**

- The bulk-rewrite fabrication demo is the moral climax; make the silent number-change subtle so the room feels how easily it slips by.
